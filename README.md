## Zomato Data Analysis

**Overview**

Analyzed Zomato restaurant dataset to extract insights on ratings, cuisines, pricing, and customer preferences.
Performed data cleaning, visualization, and exploratory analysis.


**Tools & Libraries**

Python, Pandas, NumPy, Matplotlib, Seaborn


**Steps**

Loaded and cleaned Zomato dataset
Handled missing values and formatted data
Explored relationships between cost, rating, and cuisine
Visualized data using bar charts, heatmaps, and scatter plots


**Insights**

Identified top cuisines and cities with highest-rated restaurants
Found correlation between price range and ratings
Highlighted patterns in restaurant popularity and customer preferences
